Sec, cosec & cot

  • sec x = (1)/(cosx)
  • cosec x = (1)/(sin x)
  • cot x = (1)/(tan x)
    cot x = (cos x)/(sin x)
  • sec2 x = 1 + tan2 x
  • cosec2 x = 1 + cot2 x

Addition formulae

  • sin (A+B) = sinAcosB + cosAsinB
  • sin (A-B) = sinAcosB - cosAsinB
  • cos (A+B) = cosAcosB - sinAsinB
  • cos (A-B) = cosAcosB + sinAsinB
  • tan (A+B) = (tanA + tanB)/(1 - tanAtanB)
  • tan (A-B) = (tanA - tanB)/(1 + tanAtanB)

Double angle formulae

  • sin2A = 2sinAcosA
  • cos2A = cos2A - sin2A
    cos2A = 2cos2A - 1
    cos2A = 1 - 2sin2A
  • tan2A = (2tanA)/(1 - tan2A)
